== Description ==
WordPress Custom User Registration form it has user friendly options. If you have a WordPress website but haven't sufficent user registration form. You can try it. It has some features which are important.

* When you signin your dashboard then get a form and control user role permission options. 
* When you signout from your dashboard then get a normal user registration form with Google Recaptcha validation option.     

<h2>How to use Recaptcha?</h2>
Go to this <a href="https://www.google.com/recaptcha/admin">Google Recaptcha</a>. Firstly register your site domain then create a new 'site key' and 'secret key'. See our screenshot.
I think it will help your projects for new user registration of WordPress. :)

Cool Features:
* <b>[nssUser_getForm]</b> Just put short-code in your page or a Template.
* We are trying to validate all of. 
* Multi user creation system when you signin. 

== Installation ==

Please follow this Steps:

1. Upload the plugin files to the '/wp-content/plugins/plugin-name' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
